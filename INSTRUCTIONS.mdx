## Comprendre

Dans les exercice précèdent nous avons récupérer les données `Todo` depuis la BDD. Maintenant si nous voulons gérer une `TodoList`, il va falloir gérer des fonctions de création / mise à jour.

- La première étape consiste à créer les requêtes SQL
- Exécuter ces requêtes

Nous allons reprendre la `TodoList` que nous avons vu dans les modules précédents et nous allons appeler notre base de données pour la création et mise a jour.

- Requête d’insertion

```sql
INSERT INTO Todo (title, isCompleted, createdAt, updatedAt)
VALUES ('Un Todo', false, NOW(), NOW())
```

- Requête de mise a jour

```sql
UPDATE Todo
    SET
      title = 'Todo updated',
      isCompleted = true,
      updatedAt = NOW()
    WHERE id = 10
```
